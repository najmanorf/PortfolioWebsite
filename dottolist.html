Project Overview

Problem

Working students often manage academic, work, and personal tasks across scattered tools, causing missed deadlines and unclear priorities.

Target Users

Working students who need to manage study, work, organization, and personal responsibilities in one structured workspace.

Solution

A collaborative task management platform with task categorization, priority tagging, shared boards, and progress tracking.

Product Goal

Help users organize responsibilities, prioritize tasks, collaborate clearly, and monitor progress more effectively.

Feature Showcase

Each feature was designed based on a specific user pain point identified during product discovery.

01

Category-Based Task Management

Problem

Users often track academic, work, and personal tasks separately, making responsibilities feel scattered and hard to manage.

Feature Solution

Enables users to create tasks and organize them into category lists, helping working students manage academic, work, and personal responsibilities in one unified workspace.

02

Priority Tagging & Deadlines

Problem

Users struggle to decide which tasks should be done first, causing urgent deadlines to be missed or deprioritized.

Feature Solution

Uses tags and deadlines to help users identify urgent tasks, manage time more effectively, and stay focused on the most important responsibilities each day.

03

Collaborative Task Boards

Problem

Team tasks often lack clear ownership and visibility, making collaboration less structured and harder to coordinate.

Feature Solution

A shared task board that enables multiple users to manage tasks together through shared lists, role-based coordination, and clear task ownership.

04

Progress Dashboard & Tracking

Problem

Users lack a quick way to know how much progress they made, which can reduce motivation and make productivity hard to evaluate.

Feature Solution

A visual dashboard that displays task completion status and progress insights, helping users monitor productivity and stay motivated to complete their goals.
