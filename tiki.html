Project Overview

TIKI Shipping Rates Check is a UI/UX redesign case study focused on improving the shipping cost checking experience for users. Acting as Product Strategist and UI/UX Designer, I analyzed the existing flow and interface to identify usability issues that caused confusion, friction, and inefficiency when users checked delivery rates.

I led the redesign process from problem discovery and competitive analysis to user flow, information architecture, and high-fidelity UI. Design decisions focused on clarity, error prevention, and faster task completion, resulting in a more intuitive rate-checking experience that aligns with user expectations and real-world usage patterns.

Key Improvements

Simplified Rate Check Form

A streamlined input form that guides users step by step when entering shipment details, reducing cognitive load and minimizing errors during the rate-checking process.

Clear Error & Empty States

Informative error and empty states that clearly explain what went wrong and how to fix it, helping users recover quickly without frustration.

Improved User Flow & IA

A restructured user flow and information architecture that separates input and results more clearly, making the shipping rate journey easier to understand and faster to complete.

Scannable Rate Results

A redesigned results page that presents shipping options, prices, and delivery estimates in a clean, scannable layout, allowing users to compare services and make decisions more confidently.
